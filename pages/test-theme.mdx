# Z80 Assembly Theme Test

This page tests Z80 assembly syntax highlighting with different themes.

```z80-assembly
; This is a comment
.org $8000          ; Origin directive
start:
    ld a, 42        ; Load immediate value
    ld hl, data     ; Load address
    jp main         ; Jump to main
    
data:
    db "Hello", 0   ; String with null terminator
    dw $1234        ; Word data
    
main:
    ; Number formats
    ld a, %11010011 ; Binary
    ld b, #FF       ; Hex with #
    ld c, 0x7F      ; Hex with 0x
    ld d, 123       ; Decimal
    
    ; Registers and operations
    add a, b
    sub c
    ret
```

```z80-klive
; Same code with z80-klive language
.org $8000
start:
    ld a, 42
    ld hl, data
    jp main
```
