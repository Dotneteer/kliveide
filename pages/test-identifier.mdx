# Identifier Test

Testing identifier recognition and precedence:

```z80-klive
; Keywords should be highlighted as keywords, not identifiers
ld a, b            ; 'ld', 'a', 'b' should be keywords/registers
nop                ; 'nop' should be keyword

; Pragmas should be highlighted as pragmas, not identifiers  
.org $8000         ; '.org' should be pragma
.equ MYCONST, 42   ; '.equ' should be pragma, 'MYCONST' should be identifier

; Statements should be highlighted as statements, not identifiers
.macro MYMACRO     ; '.macro' should be statement, 'MYMACRO' should be identifier
.if DEBUG          ; '.if' should be statement, 'DEBUG' should be identifier  
.endif             ; '.endif' should be statement

; Functions should be highlighted as functions, not identifiers
textof(A)          ; 'textof' should be function, 'A' should be register

; Pure identifiers should be highlighted as identifiers
myVariable         ; Should be identifier
_private_var       ; Should be identifier  
@special_name      ; Should be identifier
`quoted_name       ; Should be identifier
obj.property       ; Should be identifier
name123            ; Should be identifier
test!var           ; Should be identifier
value?check        ; Should be identifier

; Mixed usage
ld hl, myBuffer    ; 'ld', 'hl' keywords/registers, 'myBuffer' identifier
call myFunction    ; 'call' keyword, 'myFunction' identifier
jp main_loop       ; 'jp' keyword, 'main_loop' identifier
```

## Identifier formats supported:
- `myVariable` - Standard identifier
- `_private_var` - Underscore prefix/infix
- `@special_name` - At symbol prefix  
- `` `quoted_name `` - Backtick prefix
- `obj.property` - Dot notation
- `name123` - Alphanumeric
- `test!var` - With exclamation
- `value?check` - With question mark
