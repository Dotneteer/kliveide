# Getting Started with KliveIDE

This guide will help you get up and running with KliveIDE quickly.

## System Requirements

Before installing Klive IDE, ensure your system meets the following requirements:

- **Operating System**: Windows 10/11, macOS 10.15+, or Linux (Ubuntu 18.04+)
- **Memory**: 4GB RAM minimum, 8GB recommended
- **Storage**: 1GB available disk space
- **Display**: 1024x768 minimum resolution

## Installation

### Windows

1. Download the latest Windows installer from the [releases page](https://github.com/dotneteer/klive/releases)
2. Run the installer and follow the setup wizard
3. Launch Klive IDE from the Start menu

### macOS

1. Download the macOS package from the [releases page](https://github.com/dotneteer/klive/releases)
2. Open the downloaded `.dmg` file
3. Drag Klive IDE to your Applications folder
4. Launch Klive IDE from Applications

### Linux

1. Download the appropriate package for your distribution
2. Install using your package manager:
   ```bash
   # Ubuntu/Debian
   sudo dpkg -i kliveide_*.deb
   
   # Red Hat/Fedora
   sudo rpm -i kliveide_*.rpm
   ```

## First Launch

When you first launch Klive IDE, you'll see:

1. **Welcome Screen** - Overview of features and recent projects
2. **Project Templates** - Quick start templates for common scenarios
3. **Settings** - Configure your development environment

## Creating Your First Project

Let's create a simple "Hello World" project:

1. Click **"New Project"** from the welcome screen
2. Select **"Z80 Assembly Project"** template
3. Choose a project name and location
4. Click **"Create"**

Your new project will include:
- `main.asm` - Main assembly source file
- `project.json` - Project configuration
- `build/` - Output directory for compiled files
