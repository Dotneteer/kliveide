# Syntax Highlighting Comparison

This page compares Z80 assembly syntax highlighting with other languages to verify it's working correctly.

## Z80 Assembly (Custom Grammar)

```z80-klive
; Z80 Assembly code with custom highlighting
start:
    ld sp, 0xFFFF          ; Initialize stack
    ld a, 0x01             ; Load accumulator
    out (0xFE), a          ; Output to port
    jp main_loop           ; Jump to main
    
main_loop:
    in a, (0xFE)           ; Read input
    cp 0x1B                ; Compare with ESC
    ret z                  ; Return if zero
    jr main_loop           ; Continue loop
```

## JavaScript (Built-in Grammar)

```javascript
// JavaScript code with built-in highlighting
function fibonacci(n) {
    if (n <= 1) return n;
    return fibonacci(n - 1) + fibonacci(n - 2);
}

const result = fibonacci(10);
console.log(`Fibonacci(10) = ${result}`);
```

## TypeScript (Built-in Grammar)

```typescript
// TypeScript code with built-in highlighting
interface User {
    id: number;
    name: string;
    email?: string;
}

class UserManager {
    private users: User[] = [];
    
    addUser(user: User): void {
        this.users.push(user);
    }
    
    getUser(id: number): User | undefined {
        return this.users.find(u => u.id === id);
    }
}
```

## Python (Built-in Grammar)

```python
# Python code with built-in highlighting
class Calculator:
    def __init__(self):
        self.result = 0
    
    def add(self, value):
        self.result += value
        return self
    
    def multiply(self, value):
        self.result *= value
        return self
    
    def get_result(self):
        return self.result

# Usage
calc = Calculator()
result = calc.add(5).multiply(3).get_result()
print(f"Result: {result}")
```

## Plain Text (No Highlighting)

```text
This is plain text without any syntax highlighting.
It should appear in a monospace font but without colors.
Even keywords like function, class, or ld should not be highlighted.
```

## Expected Results

- **Z80 Assembly**: Should show custom highlighting with keywords in blue, comments in green, numbers in appropriate colors
- **JavaScript/TypeScript**: Should show standard syntax highlighting
- **Python**: Should show standard Python syntax highlighting  
- **Plain Text**: Should appear in monospace without any color highlighting
