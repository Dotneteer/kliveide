# Machine Types

This document summarizes the machines and models available in Klive IDE.

# ZX Spectrum 48K

Machine ID: `sp48`

Available models:
- `pal`: The original ZX Spectrum 48K model with PAL video output.
- `ntsc`: The original ZX Spectrum 48K model with NTSC video output.
- `pal-16k`: The original ZX Spectrum 48K model with PAL video output and 16K RAM.
