# Machine Types

This document summarizes the machines and models available in Klive IDE.

## ZX Spectrum 48K

Machine ID: `sp48`

Available models:
- `pal`: The original ZX Spectrum 48K model with PAL video output.
- `ntsc`: The original ZX Spectrum 48K model with NTSC video output.
- `pal-16k`: The original ZX Spectrum 48K model with PAL video output and 16K RAM.

## ZX Spectrum 128K

Machine ID: `sp128`

Available models: Only the original ZX Spectrum 128K model is available.

## ZX Spectrum +2E/+3E

Machine ID: `spp3e`

Available models:
- `nofdd`: ZX Spectrum +2E (without floppy disk drive).
- `fdd1`: ZX Spectrum +3E (with a single floppy disk drive).
- `fdd2`: ZX Spectrum +3E (with two floppy disk drives).

## ZX Spectrum Next

Machine ID: `zxnext`

Available models: Only the original ZX Spectrum Next (KS2) model is available; it is still in development.

## Cambridge Z88

Machine ID: `z88`

Available models:
- `OZ50`: Cambridge Z88 (OZ v5.0 r1f99aaae)
- `OZ47`: Cambridge Z88 (OZ v4.7)
- `OZ40`: Cambridge Z88 (OZ v4.0 UK)
- `OZ40FI`: Cambridge Z88 (OZ v4.01 SE/FI)
- `OZ30`: Cambridge Z88 (OZ v3.0 UK)
- `OZ323IT`: Cambridge Z88 (OZ v3.23 IT)
- `OZ326FR`: Cambridge Z88 (OZ v3.26 FR)
- `OZ319ES`: Cambridge Z88 (OZ v3.19 ES)
- `OZ321DK`: Cambridge Z88 (OZ v3.21 DK)
- `OZ318DE`: Cambridge Z88 (OZ v3.18 DE)
